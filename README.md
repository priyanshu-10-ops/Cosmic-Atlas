# Cosmic Atlas

Cosmic Atlas is an interactive astronomy and astrophysics learning platform designed to make exploring the universe engaging and accessible. Inspired by modern space agencies and scientific discovery platforms, it combines immersive visuals, educational content, and interactive tools to help users learn about the Solar System, deep-space objects, cosmology, space missions, and the fundamental science behind the cosmos.

## Features

* Interactive Solar System Explorer
* Deep Space Explorer (Galaxies, Nebulae, Black Holes, Exoplanets)
* Astrophysics Learning Center
* Timeline of the Universe
* Space Missions Hub
* Latest Discoveries Dashboard
* Weight on Other Planets Calculator
* Cosmic Distance Visualizer
* Star Lifecycle Simulator
* Astronomy Quizzes and Learning Resources

## Tech Stack

* React
* TypeScript
* Vite
* Tailwind CSS

## Getting Started

```bash
git clone https://github.com/dolphin-tech-max/Cosmic-Atlas-v1.git

cd Cosmic-Atlas-v1

npm install

npm run dev
```

Open `http://localhost:5173` in your browser.

## Build for Production

```bash
npm run build
```

## Preview Production Build

```bash
npm run preview
```
